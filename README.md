# MarkdownEditor

## Installation

1. Clone the repository:
    ```sh
    git https://github.com/2valide/Groupe10.git
    ```
2. Navigate to the project directory:
    ```sh
    cd Groupe10
    ```
3. Install the dependencies:
    ```sh
    npm install
    ```

## Usage

1. Start the development:
    ```sh
    npm run dev
    ```
